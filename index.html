Critical: One Tab Rule
You can only have ONE website tab open at a time during OnBase uploads to prevent system crashes. Close all other tabs before proceeding.

Upload Process

1
Preparation
Open Google Drive and locate the source file you need to upload
2
File Management
Immediately move the file to the "Uploaded Folder" to prevent duplicate work and track your progress
3
System Entry
Enter the Student UNI into OnBase and press Tab to autofill student information
4
Verification
Verify all student information is correct before completing the upload
5
Completion
Submit the upload and confirm successful processing in the system

Troubleshooting: Student UNI Issues

Follow this decision tree when encountering issues with student data entry:

IF: Issue Detected
Student name/UNI does not autofill after pressing Tab key
THEN: First Action
Open the student directory and manually search for the student by name or partial UNI
IF: Still Not Found
Manual search in the directory produces no results
THEN: Skip & Document
Skip this file and leave a detailed comment explaining the issue. Move the file to the "Exceptions" folder for supervisor review

RAF Email Processing

Follow these verification steps for each RAF (Registration Approval Form) email:

1
Check Term
Verify the academic term is correct for the registration request
2
Verify Student Status
Confirm the student is one of our grad students using the directory [grad]
3
Validate Call Number
Use Vergil course search to verify the call number matches the correct term
4
Assign to Jackie
Once all checks are complete, assign the RAF email to Jackie for final processing
